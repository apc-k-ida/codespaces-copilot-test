# codespaces-copilot-test

GitHub Copilot 研修の事前確認用リポジトリです。  
お客さまの GitHub 環境で Codespaces を起動し、Copilot が利用できるかを事前に確認できます。

## 目的

- Codespaces が起動できることを確認する
- GitHub Copilot / Copilot Chat が利用可能な状態かを確認する

## 使い方

1. このリポジトリを開き、`Code` → `Codespaces` → `Create codespace on main` を実行
2. Codespaces 起動後、VS Code 右下や拡張機能で **GitHub Copilot** / **GitHub Copilot Chat** が有効か確認
3. 任意のファイルで以下を試し、補完やチャット応答が返ることを確認
   - インライン補完（Tab で候補を受け入れ）
   - Copilot Chat への質問

## うまくいかない場合

- GitHub アカウントで Copilot ライセンスが有効か確認
- 組織ポリシーで Copilot / Codespaces が許可されているか確認
- Codespaces の再起動後に再確認
